## Accessibility

The popup component implements the following accessibility features:

- Uses `aria-label` for the close button
- Maintains proper focus management
- Provides sufficient color contrast for all variants
- Supports keyboard navigation for action buttons
- Can be dismissed via keyboard (Escape key)
