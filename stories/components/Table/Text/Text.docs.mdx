## Accessibility

The table component implements proper semantic table structure and ARIA attributes:

- Uses semantic `<table>`, `<tbody>`, and `<tr>` elements
- Includes appropriate ARIA roles: `table`, `rowgroup`, and `row`
- Provides an `aria-label` for screen readers
- First column is visually distinguished with bold text and different background
- Maintains proper color contrast for all text content
