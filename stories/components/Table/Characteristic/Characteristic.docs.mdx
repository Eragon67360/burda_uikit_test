## Accessibility

The characteristic table implements several accessibility features:

- Uses semantic table structure with `<thead>` and `<tbody>`
- Maintains clear visual hierarchy with styled headers
- Provides consistent text alignment patterns for better readability
- Uses alternating row backgrounds to improve row distinction
- Ensures proper color contrast for all text content
- Icons are properly sized and centered for visual consistency
- First column acts as row headers with bold styling
