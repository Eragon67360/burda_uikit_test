## Accessibility

The Tabs component follows the WAI-ARIA Tabs Pattern and implements the following accessibility features:

- Keyboard navigation:
  - Left/Right arrows to move between tabs
  - Up/Down arrows as alternatives
  - Automatic focus management
- Focus indicators for keyboard navigation

### Keyboard Support

`Tab` : Moves focus to the selected tab and its associated panel

`Left/Right Arrow` : Moves focus to the previous/next tab

`Up/Down Arrow` : Same as Left/Right Arrow

`Space/Enter` : Activates the focused tab
