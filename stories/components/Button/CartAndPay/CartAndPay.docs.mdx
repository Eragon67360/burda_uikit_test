## Usage

The CartAndPay component is a specialized button designed for cart and payment actions. It features an icon, customizable label, and an optional item count display.

## Accessibility

The CartAndPay component implements several accessibility features:

1. **ARIA Labels**:

   - `aria-label` for inner count badge to describe amount of items in cart

2. **Keyboard Navigation**:

- Focusable with keyboard
- Visual focus indicators
- Proper tab order

3. **States**:

   - Visible disabled state
   - Focus and hover states
   - Active state feedback

4. **Screen Reader Support**:
   - Meaningful cart count announcements
   - Icon is properly hidden from screen readers
   - Clear button purpose indication
