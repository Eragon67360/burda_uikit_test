# ButtonIcon Component

The ButtonIcon component is a versatile button that displays an icon. It supports various states and can be customized for accessibility.

## Accessibility

The ButtonIcon component supports accessibility through the `ariaLabel` prop. This label is crucial for screen readers to provide context about the button's function.

### Example

```html
<button type="button" aria-label="Close modal" class="...">
  <!-- Icon -->
</button>
```

## Customization

The component supports different:

- Variants (Small, Big)
- Background colors (neutral-100, neutral-200)
- Icons from the system icon registry
- Disabled states
