## Accessibility

The ButtonIcon component supports accessibility through the `ariaLabel` prop. This label is crucial for screen readers to provide context about the button's function.

### Example

```html
<button type="button" aria-label="Close modal" class="...">
  <!-- Icon -->
</button>
```
