## Accessibility

The SelectionList component follows these accessibility best practices:

- Uses semantic HTML with proper `<input type="radio">` elements
- Groups related radio buttons using the same `name` attribute
- Supports standard form submission

### Keyboard Navigation

- Use <kbd>Tab</kbd> to focus the radio group
- Use <kbd>↑</kbd> / <kbd>↓</kbd> or <kbd>←</kbd> / <kbd>→</kbd> to navigate between options
- Use <kbd>Space</kbd> to select the focused option
