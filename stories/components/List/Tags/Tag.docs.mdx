## Accessibility

- Tags use semantic HTML elements for proper structure
- Text content is readable and has sufficient contrast
- Interactive elements (if added) should be keyboard accessible
- Group container provides proper overflow handling for mobile scrolling
- Icons are decorative and properly hidden from screen readers
