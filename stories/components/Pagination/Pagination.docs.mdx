### Component Markup

  language="html"

### Register JavaScript globally

<Source
  language="html"
  dark
  code={`
<script src="pagination.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', () => {
    const pagination = createPagination({
      currentPage: 1,
      totalPages: 10,
      onPageChange: (page) => {
        console.log('Page changed to:', page);
      }
    });
    document.querySelector('#pagination-container').appendChild(pagination);
  });
</script>
`}
/>

## Accessibility

The pagination component implements the following accessibility features:

- Current page button has `aria-current="page"` attribute
- Navigation buttons have descriptive `aria-label` attributes
- Disabled buttons have the `disabled` attribute
- Focus indicators are visible for keyboard navigation
