## Accessibility

The navigation buttons implement the following accessibility features:

- Proper `aria-label` describing the button's action ("next slide" or "previous slide")
- Disabled state properly conveyed using both `disabled` and `aria-disabled` attributes
- Adequate touch target size (44px minimum) on all devices
- Icon-only buttons have descriptive labels for screen readers
