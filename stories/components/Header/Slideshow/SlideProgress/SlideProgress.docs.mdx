## Accessibility

The progress indicator implements the following accessibility features:

- Interactive elements (steps and play/pause button) are keyboard-focusable
- Touch targets meet minimum size requirements (44px on mobile)
- Progress animations respect user's reduced motion preferences

### Keyboard Navigation

- `Tab`: Navigate between interactive elements
- `Space/Enter`: Activate focused element (step navigation or play/pause)
- Focus is properly trapped within the component
- Focus indicators are visible and maintain sufficient contrast
