### Register JavaScript globally

<Source
  language="html"
  dark
  code={`
<script src="language-dropdown.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', () => {
    const dropdown = createLanguageDropdown({
      options: [
        { code: 'en', name: 'English', icon: 'england' },
        { code: 'fr', name: 'Français', icon: 'france' }
      ],
      selectedLanguage: 'en',
      label: 'Language'
    });
    document.querySelector('#language-selector').appendChild(dropdown);
  });
</script>
`}
/>

## Accessibility

The language dropdown implements the following accessibility features:

- Uses proper ARIA attributes (`aria-expanded`, `aria-haspopup`, `aria-controls`)
- Implements radio group pattern for language selection
- Supports keyboard navigation
- Provides visual feedback for interactive states
- Maintains focus management
- Uses semantic HTML structure
