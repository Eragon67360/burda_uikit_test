## Usage

The Badge component is used to display status information or notifications, typically positioned absolutely relative to its parent container.

### Component Markup

<Source
  language="html"
  dark
  code={`
<!-- Basic Badge -->
<div 
  class="absolute top-0 left-0 p-2 aspect-square bg-primary-interaction rounded-full flex items-center justify-center"
  role="status"
  aria-label="Tipp"
  style="min-width: 42px; min-height: 42px;"
>
  <div class="font-bold text-copy-small">
    Tipp
  </div>
</div>

<!-- Secondary Badge -->

<div
  class="absolute top-0 left-0 p-2 aspect-square bg-secondary-interaction rounded-full flex items-center justify-center"

  aria-label="Status"

>
  <div class="font-bold text-copy-small">Status</div>
</div>
`} />

## Accessibility

The Badge component implements the following accessibility features:

- Uses `role="status"` to indicate its purpose
- Includes `aria-label` for screen readers
- Maintains sufficient color contrast for text visibility
- Provides meaningful text alternatives through `aria-label` when badge text is not descriptive enough
